== Description ==

This plugin adds a "Translate Theme" page to the administrator's settings menu and theme page

== Installation ==

1.  Upload wp-translate-theme-jun/ directory to the wp-content/plugins directory.
3.  Browse to 'settings' -> 'translate' and have at it.
